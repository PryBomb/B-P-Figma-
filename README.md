# B&P design 
Home page website design 

Space Landing Page
A sleek, modern, and immersive landing page design featuring a Glassmorphic UI and vibrant gradient aesthetics. This project showcases a high-performance digital environment designed for the next generation of web experiences.

Project Overview
The Space landing page was designed with a focus on visual depth and minimalism. By utilizing frosted-glass effects and a vibrant color palette, the interface creates an inviting yet professional atmosphere for users to explore.

Design Highlights
Glassmorphism: A centerpiece hero card featuring a semi-transparent, blurred background for a modern "frosted" look.

Vibrant Gradients: A dynamic mix of deep purples, magentas, and soft orange glows to simulate a cosmic, high-energy environment.

Clean Typography: Focused on readability and hierarchy, ensuring the "Experience a modern world" message is the primary focal point.

Responsive Layout: Designed with scalability in mind, from the minimalist navigation bar to the centered CTA button.

Tech Stack & Tools
Design: Figma

Assets: Custom vector shapes and Gaussian blur effects.

Theme: Dark Mode / Neo-Futurism.
