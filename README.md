# B&P design 
Home page website design 

A high-fidelity landing page design featuring a **Glassmorphism** aesthetic. This project explores the intersection of futuristic visuals and functional UI, utilizing deep cosmic gradients and frosted-glass elements to create an immersive user experience.

## 🚀 Vision
The **Space** project is designed to represent a gateway to modern innovation. By blending vibrant purples and oranges with a clean, minimalist layout, it provides a sophisticated backdrop for tech-forward brands.

## ✨ Key Features
* **Glassmorphic Hero Card:** Uses a semi-transparent background with a high-intensity backdrop blur.
* **Neon Glow Borders:** Subtle glowing strokes around the main container to separate it from the cosmic background.
* **Minimalist Navigation:** A simplified top-bar layout focused on user flow and accessibility.
* **Responsive Typography:** Optimized hierarchy for high-impact headlines and readable body text.

## 🛠️ Built With
* **Figma** – Interface design and prototyping.
* **Auto-Layout** – For consistent spacing and responsive scaling.
* **Custom Gradients** – Multi-stop mesh gradients for the "Space" background.
